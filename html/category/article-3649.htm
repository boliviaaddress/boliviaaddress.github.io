选择排序的java实现

  • 首页
  • 文章分类
  • 正文

众所周知排序算法是面试中的重要考察点，也可以看一个人基础知识掌握的是否牢固。这篇文章带大家来看看排序算法中很重要的一种排序：选择排序（从小到大排序）。

基本思路是：在有序部分（刚开始有序部分的长度为0）后面的数中选择最小的数，交换排好序部分的后一个数和最小的数，有序部分的长度加一，直到有序部分的长度等于数组的长度，排序结束。

图示：

java实现代码：

public static void selectionSort(int []arr) 	{ 		if(arr==null||arr.length<2)//传入空指针或数组的大小小于2直接返回 			return ; 		for(int i=0;i<arr.length-1;i++) 		{ 			int minNumber=i; 			for(int j=i+1;j<arr.length;j++)//找到i+1到arr.length中的最小值 			{ 				if(arr[j]<arr[minNumber]) 					minNumber=j; 			} 			swap(arr,i,minNumber);//交换arr[i]和arr[minNumber],有序部分加一 		} 	}

分析时间复杂度：两层for循环，时间复杂度为O(n^2);只用了有限几个变量，空间复杂度为O(1)。
