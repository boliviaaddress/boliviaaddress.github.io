学习过高数知道，负数开方后是个复数，怎么得到呢？

我们知道math库里有个sqrt()函数，但这里不能用这个，要使用cmath库里的sqrt()，毕竟是专门来处理复数的；math库主要是处理int和float类型的数据。

>>> import cmath >>> cmath.sqrt(-1) 1j 
>>> import math >>> math.sqrt(2) 1.4142135623730951
  • 布尔类型

非0即1

怎么把一个数据转化成bool值？用bool()

>>> bool(-1) True >>> bool(0) False >>> bool(1) True >>> bool(0.000000000001) True

定义和初始化一个bool型变量：

>>> a=True >>> a True >>> type(a) <type 'bool'>
